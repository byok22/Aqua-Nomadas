# Proyecto Piloto de Aguas Abiertas - Santa Maria del Oro

Este repositorio contiene la implementacion operativa y comercial del piloto de entrenamiento de aguas abiertas.

## Objetivo del piloto
- Lugar: Santa Maria del Oro.
- Fecha objetivo: octubre 2026.
- Duracion: 1 dia.
- Capacidad: 10 participantes.
- Perfil: nadadores intermedios de alberca.
- Distancia objetivo piloto: hasta 3 km.
- Vision escalable: habilitar ediciones futuras hasta 5 km, solo despues de validar seguridad y operacion.

## Que incluye este sistema
- Manual operativo general.
- Manual de seguridad (guardavidas, lancha, kayaks).
- Roles y responsables.
- Itinerario completo del dia del piloto.
- Manual de contenido para Instagram (imagen + video).
- Plan de promocion Instagram + WhatsApp.
- Plantillas: checklist, consentimiento y presupuesto.

## Orden recomendado de uso
1. Leer `01_Operativo/Manual_Operativo_General.md`.
2. Ajustar `03_Seguridad/Manual_Seguridad_Piloto.md` con proveedores reales.
3. Asignar equipo en `01_Operativo/Roles_y_Responsables.md`.
4. Ejecutar simulacro con `04_Itinerarios/Itinerario_Piloto_Dia_1.md`.
5. Publicar contenido siguiendo `02_Marketing/Manual_Instagram_Paso_a_Paso.md` y `02_Marketing/Plan_Promocion_Instagram_WhatsApp.md`.
