# Checklist Dia del Evento

## A) Seguridad
- [ ] Guardavidas presentes y acreditados.
- [ ] Botiquin completo.
- [ ] Oxigeno funcional y revisado.
- [ ] Comunicacion interna operativa.
- [ ] Punto de salida identificado y comunicado.

## B) Operacion
- [ ] Itinerario impreso o digital para staff.
- [ ] Roles asignados y confirmados.
- [ ] Briefing interno realizado.
- [ ] Material de registro listo.

## C) Participantes
- [ ] Lista de asistencia.
- [ ] Consentimientos firmados.
- [ ] Contacto de emergencia validado.
- [ ] Entrega de gorra.

## D) Ejecucion en agua
- [ ] Briefing de seguridad impartido.
- [ ] Grupo segmentado por ritmo.
- [ ] Pausas tecnicas realizadas.
- [ ] Control de fatiga en cada bloque.

## E) Cierre
- [ ] Debrief tecnico realizado.
- [ ] Hidratacion/snack entregado.
- [ ] Incidencias registradas.
- [ ] Foto/evidencia del cierre.
- [ ] Retroalimentacion del equipo.
