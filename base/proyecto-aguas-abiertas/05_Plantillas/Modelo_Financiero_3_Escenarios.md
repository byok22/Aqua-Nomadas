# Modelo Financiero en 3 Escenarios (Piloto 10 personas)

Moneda base: MXN
Formato: piloto de 1 dia, nivel intermedio, hasta 3 km.

## Supuestos de costos
- Costos fijos base estimados: 18,000
- Costos variables por participante: 180
- Cupo: 10

## Escenario 1 - Conservador
Supuesto: costos suben y cierre comercial es mas lento.
- Costos fijos: 20,000
- Variable por participante: 220
- Margen objetivo por participante: 250

Formula:
Precio = (Fijos/10) + Variable + Margen
Precio = (20,000/10) + 220 + 250 = 2,470

Precio sugerido conservador: 2,500
Ingreso total a cupo lleno: 25,000
Utilidad estimada: 25,000 - (20,000 + 2,200) = 2,800

## Escenario 2 - Base
Supuesto: operacion segun plan actual.
- Costos fijos: 18,000
- Variable por participante: 180
- Margen objetivo por participante: 320

Precio = (18,000/10) + 180 + 320 = 2,300

Precio sugerido base: 2,300
Ingreso total a cupo lleno: 23,000
Utilidad estimada: 23,000 - (18,000 + 1,800) = 3,200

## Escenario 3 - Agresivo
Supuesto: mejor control de costo y alto valor percibido.
- Costos fijos: 17,000
- Variable por participante: 170
- Margen objetivo por participante: 530

Precio = (17,000/10) + 170 + 530 = 2,400

Precio sugerido agresivo: 2,400
Ingreso total a cupo lleno: 24,000
Utilidad estimada: 24,000 - (17,000 + 1,700) = 5,300

## Precio definitivo recomendado
- Precio publico: 2,300
- Preventa temprana (primeros 4 lugares): 2,100
- Ultimos lugares (despues de 70% de cupo): 2,400

## Politica de cobro recomendada
- Anticipo: 50% para reservar.
- Liquidacion: 72 h antes.
- Si no liquida en plazo: liberar lugar.

## Punto de equilibrio
Punto de equilibrio en participantes:
PE = Costos fijos / (Precio - Variable)
PE = 18,000 / (2,300 - 180) = 8.49

Se requieren 9 participantes para cubrir costos con escenario base.

## Regla operativa de seguridad financiera
No ejecutar evento con menos de 9 confirmados de pago, salvo decision estrategica explicita.
